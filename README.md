# Violet.js - A Twitter Library for Firefox OS

## Setup

```sh
$ git clone git@github.com:masawada/violet.git
$ git submodule init
$ git submodule update
$ npm install
```

## Test

```sh
$ cd test
$ ../node_modules/.bin/bower install
```

install and run test app in firefox os simulator.

test app is `./test/`.
